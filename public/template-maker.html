포토부스 테마 템플릿 생성기

각 레이아웃에서 사진이 나오는 위치와 꾸밈 영역을 확인하고, 템플릿을 다운로드하세요.

템플릿 사용 방법
1. 아래에서 원하는 레이아웃의 템플릿 PNG를 다운로드합니다.
2. Photoshop / Procreate / Figma 등에서 열어 초록 영역(#00C500) 위에 꾸밈 요소를 그립니다.
3. 초록 픽셀은 앱에서 투명하게 처리되어 카메라 영상이 보입니다 — 초록 위에 그리면 사진 위에 겹쳐집니다.
4. 완성된 PNG를 public/themes/{테마명}/{레이아웃폴더}/{슬롯번호}.png 경로에 저장합니다.

파일 경로 예시 — 테마 "spring", 레이아웃 1×4, 슬롯 1번:
public/themes/spring/1_4/1.png

크로마키 색상 : rgb(0, 197, 0) / #00C500 — 이 색이 투명해집니다.
